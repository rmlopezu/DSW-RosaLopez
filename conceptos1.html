Etiquetas HTML

En esta página se presentan contenidos debidamente redactados sobre de la temática escogida para la construcción del sitio web que cuentan con imágenes de apoyo. Las lecturas e imágenes están debidamente citadas con norma APA. (Ver referencias en la segunda sección)


Etiqueta <section>

Es una sección genérica en un documento que se utiliza para establecer el contenido que hace parte de un esquema. Ayuda a esquematizar como en el índice del contenido de un libro; un tema común y subsecciones relacionadas, es decir que su principal función es estructurar semánticamente un documento. (w3schools. HTML <section> Tag, 2012)

(w3schools. HTML <section> Tag, 2012)


Etiqueta <aside>

Define algún contenido además del contenido principal, está relacionado indirectamente con el contenido circundante. Suelen representarse como barras laterales o como inserciones.(w3schools. HTML <aside> Tag, 2012)

(w3schools. HTML <aside> Tag, 2012)
